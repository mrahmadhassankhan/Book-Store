# MERN Bookstore Application

## Description
A MERN stack-based web application for managing a bookstore. This project provides functionalities for both administrators and students, allowing administrators to manage users and books, while students can browse and explore available books.

## Technologies Used
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB

## Installation
1. Clone the repository: `git clone https://github.com/mrahmadhassankhan/Book-Store.git`
2. Navigate to the `server` directory
3. Create a `.env` file and add the following environment variables:
   ```plaintext
   URI="mongodb://localhost:27017/bookstore"
   Admin_Pass="admin"
   PORT="3001"
   ADMIN_KEY="Studentkeyishere"
   STUDENT_KEY="adminkeyishere"

4. Run `npm install` to install dependencies for the server
5. Navigate to the `client` directory
6. Run `npm install` to install dependencies for the client

## Usage
1. Start the server: `npm start` (inside the `server` directory)
2. Start the client: `npm run dev` (inside the `client` directory)
3. Open your web browser and visit `http://localhost:3000` to access the application
